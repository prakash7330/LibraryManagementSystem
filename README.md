# 📚 Library Management System

A full-stack Library Management System built with Java, Spring Boot, MySQL, HTML and CSS.

## 🔗 Tech Stack
- **Backend:** Java, Spring Boot, Spring Data JPA
- **Database:** MySQL
- **Frontend:** HTML, CSS, Thymeleaf
- **Console Version:** Core Java, JDBC, DAO Pattern

## ✨ Features
- 📖 Book Management — Add, Search, Edit, Delete books
- 👥 Member Management — Register and manage members
- 🔄 Borrow & Return — Issue books and track returns
- 📊 Dashboard — Real-time stats on home page

## ⚙️ Setup Instructions
1. Install Java JDK 17, MySQL, Maven
2. Create database: `library_db`
3. Run `database.sql` in MySQL
4. Update `application.properties` with your MySQL password
5. Run `.\mvnw spring-boot:run`
6. Open `http://localhost:8080`

## 📸 Pages
- Home — `http://localhost:8080/`
- Books — `http://localhost:8080/books`
- Members — `http://localhost:8080/members`
- Borrow/Return — `http://localhost:8080/borrow`
